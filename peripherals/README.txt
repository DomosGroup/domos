This folder contains firmware code for every peripheral officially supported by Domos.
Naming conventions are as follows for every firmware file in this folder:
  XXXXYYY_ZZZ
XXXX: peripheral type.
YYY:  peripheral model number.
ZZZ:  peripheral firmware version.

E.G.: TEMP001_100 is the firmware source for the temperature peripheral number 001. Its version is 1.0.0.
